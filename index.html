🚀 Cara Pakai

Mulai dalam 5 Menit

Ikuti langkah-langkah berikut untuk menjalankan Fosh Bot di WhatsApp-mu.

1

Clone Repository

git clone https://github.com/nargeloooooooooooooooo/fosh-bot-wa.git

Download kode sumber bot ke komputer-mu.

2

Install Dependencies

npm install

Install semua library yang dibutuhkan.

3

Konfigurasi

Edit src/config.js

Ganti owner.number dengan nomor WhatsApp-mu.

4

Jalankan & Scan

npm start

Scan QR Code di terminal dengan WhatsApp.
